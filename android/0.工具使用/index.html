工具使用

参考教程版本：

应用运行

模拟器

  1. 创建Android虚拟设备(virtual device)：==Tools→AVD Manager==
  2. 选择硬件设备、系统镜像和虚拟机设备
  3. 快捷键Control+R运行应用

物理设备

Windows系统需要安装adb(Android Debug Bridge)驱动。

打开设备的USB调式模式。

代码补全

Android Studio的代码补全功能可以简化编程，节省时间。

编写程序时，摁下Enter、Tab、Return键或鼠标点击即可使用该功能。

自动生成类的getter和setter方法

  1. 配置识别成员变量的m前缀：==File→Settings→Editor→Code Style→Java→Code Generator==，naming表中设置前后缀
  2. java文件，右键构造方法后方区域，选择==Generator→Getter and Setter==

日志跟踪

Android的android.util.Log类能够向系统级共享日志中心发送日志信息。

public static int d(String tag,String msg)

d代表debug，第一个参数是日志来源，通常以类名为值的TAG常量传入。

第二个参数是日志的具体内容。

LogCat

应用运行时，可以使用LogCat工具查看日志。
